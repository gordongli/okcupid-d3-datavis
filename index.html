License

“Data scraped from public profiles on www.okcupid.com on 06/30/2012. Permission to use this data was obtained from OkCupid president and co-founder Christian Rudder under the condition that the dataset remains public.” (okcupid_codebook.txt, Kim and Escobedo-Land)

Description

A data set consisting of user profile data for 59,946 San Francisco OkCupid users (a free online dating website) from June 2012. Each row represents the data for one user profile, and columns include user information and lifestyle variables such as sex, sexual orientation, age, height, ethnicity, education, income, religion, drinking habits, and smoking habits, along with text responses to the 10 essay questions such as “My Self Summary”.

Motivation

I chose this dataset because the possibility of quantifying dating in the digital world is interesting to me. I had seen some intriguing data visualizations on the OkCupid blog in the past, and was pleased to discover this dataset so that I too may try to discover some insights from it. The dataset is also sampled from San Francisco, so it may be of particular relevance to the students at the University of San Francisco taking the Data Visualization course for which this project is being done.

Aims

I hope to learn some interesting things about human culture, psychology, and behavior in general and specifically concerning courtship from this dataset. I would like to look for unexpected relationship between the various variables provided in user profile data. I would also like to see how people of different groups describe themselves in different ways in their text responses.
